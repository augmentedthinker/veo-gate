Veo Gate

A minimal gated web app for Gemini / Veo video generation. Enter the shared 4-digit access code, describe the video you want, and the backend handles the hidden API call using a server-side Gemini key.

This is a lightweight personal gate, not serious authentication.
Ready.
The frontend never sees the Gemini API key. It only talks to backend endpoints that validate the code and perform Veo generation server-side.
