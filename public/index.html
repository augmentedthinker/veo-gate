Gemini Media Gate

A gated Gemini media app for testing both image and video generation with a hidden server-side API key. Image mode is the faster, cheaper debug lane. Video mode is the higher-cost premium lane.

This is a lightweight personal gate, not serious authentication.
Ready.
The frontend never sees the Gemini API key. It only talks to backend endpoints that validate the code and perform generation server-side through Vercel functions.
